# 美团出行助手 · meituan-coupon-traffic

> 【美团官方】让每次出行都更划算——一键领取美团机票 & 火车票专属优惠券，支持每日定时自动领取，领完立即通知你。
---

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 🎫 **一键领券** | 覆盖美团机票、火车票优惠券，领取秒到账 |
| 📋 **历史查询** | 查询指定日期或日期范围内的历史领券记录 |
| 🔐 **内置认证** | 无需额外 Skill，手机号 + 短信验证码即可完成美团账号登录 |
| ⏰ **定时自动领券** | 支持每天指定时间自动领券，领完推送通知，无需手动触发 |

---

## 🚀 快速开始

### 触发词

直接对 AI 说以下任意内容即可触发：

```
领火车票券 / 领机票券 / 领出行红包 / 出行优惠 / 我要领券 / 领权益
```

### 使用流程

```
1. 说「领火车票券」或「领机票券」
   ↓
2. 首次使用：阅读并同意《Skill服务使用规则》
   ↓
3. 输入手机号 → 收到短信验证码 → 完成美团账号认证
   ↓
4. 自动领券，展示券详情（名称、面额、有效期、使用链接）
   ↓
5. 可选：设置每日定时自动领券（默认早上 10 点）
```

> ✅ 认证信息仅保存在本地设备，不会上传至任何服务器。
